# Vite playground
This project was created with Vite.

![](https://i.imgur.com/F9btt4j.png)

A playground to play with:

- React
- React router dom
- Typescript
- Axios
- Tanstack Query
- React Hook Form
- Tailwind
- Json Server

## Project initialization

1. ``` $ git clone https://github.com/aldaricse/react-query-crud.git ```
2. ``` $ cd react-query-crud ```
3. ``` npm i ```
4. ``` npm run dev ```
4. ``` npm run back ```
